These packages will auto-detect any installed suitable back-end compilers.
Under Windows, Visual Studio 2015 and Visual Studio 2017 will be auto-detected. You can also download a pre-packaged Prepackaged 32/64 bit TDM version if you do not have/wish to use Visual Studio.
Under Linux, GCC (C++ support required) will be auto-detected if installed.

Consult the installation instructions for more information.
