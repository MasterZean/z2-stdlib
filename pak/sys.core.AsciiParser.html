SkipWhitespace

val SkipWhitespace;

Brief

If set to `true`, every time an entity is consumed, whitespace following said entity is also consumed. If set to `false`, whitespace is not consumed and. It can manually be consumed with `EatSpaces`.

SkipComments

val SkipComments;

Brief
