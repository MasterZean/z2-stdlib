sys.core.ConvertUtf8 IntAsText
=
## Brief
Writes a `sys.core.lang.Int` to a buffer as an Utf8 text.

The buffer `Length` is updated to the number of code units written.

### param buffer
the output buffer
### param data
the value to write
### param base
the base to use
